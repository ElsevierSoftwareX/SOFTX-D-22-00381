# Argnnotator tool
![version](https://img.shields.io/badge/version-0.9.4-blue)
![last-update](https://img.shields.io/badge/last_update-7/21/2022-orange)

## Dependencies
The implemented solution depends on or make use of the following libraries:
- JDK 16
- Snake YAML 1.9
- JSON Java 20210307
